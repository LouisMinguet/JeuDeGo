Ce programme permet de jouer rapidement et facilement au jeu de GO avec un ami.
Vous avez la possibilité de changer des options comme le temps et les handicaps.
Ce jeu utilise la règle Française/Chinoises.

Ce projet est réalisé dans le cadre de mes études.

	----- [Pour lancer le jeu] -----

Il suffit de lancer l'archive JAVA GO.jar

Sinon, on se place dans le dossier, on éxecute la commande "javac *.java" puis "java Menu" pour accéder au launcher.

	----- [Programmes] -----

> Programmes pour le jeu :

     1. Jeu.java 	: Contient le programme de jeu avec affichage du plateau et gestion des règles.
     2. Groupe.java	: Contient la gestion des groupes avec les différentes actions possibles.
     3. Pierre.java	: Contient la gestion des pierres avec leurs coordonnées.
     4. Historique.java : Contient la gestion de l'historique,

> Programmes pour le menu :

     1. Menu.java 	  : Contient le menu de lancement du jeu permettant de jouer.
     2. PartiePerso.java  : Contient le menu permettant de gérer les parties personnalisées.
				Vous pouvez alors régler la taille de la grille, le nombre de handicaps et la gestion du temps.

	----- [AVANCEMENT - 100%] -----
